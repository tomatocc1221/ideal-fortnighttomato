# 2026-05-28 开发日志

## 完成事项

### 球员档案卡片重构
- **卡片3:4固定比例**：统一卡片宽高比，避免图片尺寸不一致导致卡片高度参差
- **全息扫光动画**：卡片hover时呈现从左到右的全息光带扫过效果
- **号码水印**：卡片背景展示球员号码的巨型半透明水印
- **呼吸光晕**：卡片边缘有周期性明暗变化的光晕
- **毛玻璃卡片分组**：球员按位置（前锋/中场/后卫/门将）分组，每组带毛玻璃质感标题
- **错层入场动画**：卡片依次错开延迟入场，不再是同时出现
- **移除数据徽章**：去掉了卡片上的"数据"标签

### 背景图系统升级
- **图片降级链**：`webp → png → jpg` 三级兜底，浏览器不支持 WebP 时自动降级
- **独立 imgFallback 函数**：将降级逻辑抽成独立函数，避免代码重复
- **修复移动端背景不显示**：之前移动端图片加载失败没有兜底
