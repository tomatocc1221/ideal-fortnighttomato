# 今日说法足球俱乐部网站

## 项目信息

- **球队**：今日说法足球俱乐部（2025年3月30日成立于重庆溉澜溪体育公园）
- **技术栈**：纯 HTML + CSS + JS，无框架，无构建工具
- **设计**：黑金主题（#0d0d0d 底色，#c9a961 金色点缀），简约高级风格
- **部署**：目标 GitHub Pages（尚未上线）
- **用户**：编程初学者

## 文件结构

```
index.html          — 唯一页面（单页滚动，6个区块）
css/style.css       — 全局样式（响应式，黑金主题）
js/main.js          — 交互逻辑 + 示例数据
images/logo.jpg     — 队徽（3056×3254）
docs/               — 标准文档
devlog/             — 开发日志
```

## 工作约定

1. **改代码前先确认**：涉及功能变更时，先和用户确认需求再动手
2. **每次改动后更新日志**：在 `devlog/YYYY-MM-DD.md` 中记录完成事项
3. **保持简约**：不引入不必要的库、框架、抽象。三行重复代码好过一次过早抽象
4. **先数据后样式**：修改内容优先改 JS 数据数组或 HTML 文案，不要一上来就调 CSS
5. **每次只改一个点**：改完验证，确认没问题再进行下一步
6. **用户是初学者**：解释技术概念时用类比，给出具体文件路径和行数
