## 1. 如何访问网站

### 手机端
- 浏览器打开网站链接（队长会在微信群发布）
- **建议添加到主屏幕**：
  - iPhone：Safari 打开 → 底部「分享」→「添加到主屏幕」
  - Android：Chrome 打开 → 右上角「⋮」→「添加到主屏幕」

### 电脑端
- 浏览器打开网站链接即可
- 推荐 Chrome 或 Edge 浏览器

## 2. 认识首页

打开网站后你会看到：

| 区域 | 说明 |
|------|------|
| **顶部导航栏** | 固定在屏幕顶部，点击可快速跳转到各板块 |
| **Hero 大屏** | 队徽 + 队名 + "用脚说话，以球会友" |
| **球队介绍** | 建队信息 + 4 个统计卡片 |
| **队员阵容** | 全体队员卡片，点击可翻面查看详情 |
| **赛程战果** | 倒计时 + 最新战报 + 即将开赛 |
| **精彩瞬间** | 比赛照片轮播 + 相册浏览 |
| **战术板** | 3-3-1 阵型图 |
| **底部** | 联系方式 + 管理后台入口 |

手机端导航会收起到汉堡菜单（左上角三道杠），点击展开。

## 3. 查看队员阵容

1. 点击导航栏「队员阵容」或向下滚动到阵容区域
2. 队员按位置分组：教练、门将、后卫、中场、前锋
3. **点击卡片**可以翻转查看背面详细信息（位置、简介、身高体重、擅长位置标签）
4. 背面点击「查看详情」打开完整信息弹窗
5. 关闭弹窗：点击「← 返回」、点背景遮罩、或按 Esc 键

> 队长 (#22 李泽) 的卡片有金色边框高亮。

## 4. 赛程战果与倒计时

### 倒计时栏
- 显示最近一场即将开赛的比赛
- **倒计时**精确到秒：天、时、分、秒
- 比赛进行中时显示「比赛进行中」+ 剩余时间
- 报名按钮位于倒计时下方（详见第 5 节）

### 最新战报（左侧）
- 显示最近 3 场比赛结果
- 包含比分、胜负标记、进球队员、助攻队员
- 「查看全部战绩」链接可进入完整赛程页面

### 即将开赛（右侧）
- 显示未来 3 场比赛
- 包含日期、对手、时间、场地、球衣颜色

## 5. 比赛报名 ⭐

这是最常用的功能。每次比赛前需要报名确认参加。

### 报名窗口
- **开放时间**：比赛日前 4 天的晚上 9:00
- **截止时间**：比赛日前 2 天的中午 12:00
- 例如：周日比赛 → 周三晚 9 点开放报名，周五中午 12 点截止

### 报名步骤

1. 在首页赛程区域，倒计时栏下方找到报名按钮
2. 报名未开放时，按钮显示倒计时「距报名 X 天 XX 时 XX 分 XX 秒」
3. 开放后按钮变为金色「✋ 立即报名」，点击打开报名面板
4. **验证身份**：
   - 在下拉列表中选择你的名字
   - 输入你的个人 PIN 码
   - 点击「验证身份」
5. **选择报名类型**：
   - 「立即报名（正选）」— 正选名额（共 13 人）
   - 「立即报名（候补）」— 候补名额（共 4 人，正选满后自动进入候补）
6. 报名成功后显示「已报名 · 正选」或「已报名 · 候补」

### 请假

如果报名后有事不能参加：

1. 在报名面板中点击「我要请假」
2. 选择请假原因：临时有事 / 身体不适 / 工作加班 / 家庭事务 / 其他
3. 确认后自动取消报名，候补队列第一位自动递补为正选

### 注意事项
- 正选 13 人，候补 4 人，先报先得
- 截止前 2 小时若报名人数不足 13 人，会出现红色提醒
- 报名记录可在截止前随时修改

## 6. MVP 投票（金球奖）

每场比赛结束后，正选队员可以为表现最好的队友投票。

### 谁可以投票
- 只有该场比赛的**正选报名队员**可以投票
- 每人只能投一次，提交后**不可修改**

### 投票步骤

1. 在最新战报中找到对应的比赛，点击「MVP 投票」按钮
2. 验证身份（选择名字 + 输入 PIN 码）
3. 选择排名：
   - 第一名（3 分）
   - 第二名（2 分）
   - 第三名（1 分）
4. 同一人不能出现在多个排名中
5. 确认投票内容后提交

### 查看结果
- 比赛结果右上方显示「★ MVP: 队员名」
- 鼠标悬停可查看前三名票数
- 累计 MVP 次数会计入个人详情页的 MVP 统计

## 7. 精彩瞬间与相册

### 轮播图
- 首页「精彩瞬间」区域自动轮播比赛照片
- 点击左右箭头手动切换
- 手机端可左右滑动
- 点击任意照片进入全屏查看

### 相册浏览
- 点击导航栏「精彩瞬间」打开相册
- 照片按比赛分组（如「今日说法 vs XX 队」）
- 「比赛瞬间」相册收录所有比赛照片
- 点击照片放大查看，点击背景关闭

## 8. 战术板

首页「战术板」区域展示球队 3-3-1 阵型图：

- 橙色 = 门将
- 蓝色 = 后卫 (3人)
- 绿色 = 中场 (3人)
- 红色 = 前锋 (1人)

此为静态展示，仅供参考。

## 9. 完整赛程页面

点击导航栏「赛程战果」或首页「查看全部战绩」进入独立赛程页面。

### 功能
- **赛季筛选**：下拉框选择赛季，数据自动切换
- **统计面板**：总场次、胜、平、负统计
- **分页浏览**：每页 15 场比赛，底部翻页
- **即将开赛**：所有未开赛的比赛列表

## 10. 常见问题

### Q: 忘记 PIN 码怎么办？
联系队长重置。PIN 码用于验证身份，确保是本人操作。

### Q: 报名后怎么知道是否报名成功？
报名面板中显示你的报名状态（正选/候补），同时按钮上的数字会更新。

### Q: 手机上看加载很慢？
首次访问可能需要几秒加载。之后每次打开会使用缓存数据，基本秒开。

### Q: 网站支持哪些浏览器？
Chrome、Safari、Edge 均可。微信内置浏览器也可正常使用。

### Q: 发现 bug 或功能建议？
告诉队长或副队长，他们会反馈给开发同学。
