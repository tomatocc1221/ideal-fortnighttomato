1. 如何访问网站

手机端

  • 浏览器打开网站链接（队长会在微信群发布）
建议添加到主屏幕（像 App 一样使用）：
iPhone：Safari 打开 → 底部「分享」→「添加到主屏幕」
Android：Chrome 打开 → 右上角「⋮」→「添加到主屏幕」

电脑端

  • 浏览器打开网站链接即可
  • 推荐 Chrome 或 Edge 浏览器

2. 认识首页

打开网站后你会看到以下区域：

区域说明
顶部导航栏固定在屏幕顶部，点击各链接可快速跳转
Hero 大屏队徽 + 队名 + "用脚说话，以球会友"
球队介绍建队信息 + 4 个统计卡片（建队年份、城市、主场、班底）
队员阵容全体队员卡片，点击可翻面查看详情
赛程战果倒计时 + 最新战报 + 即将开赛 + 报名按钮
精彩瞬间比赛照片轮播 + 相册浏览
战术板3-3-1 阵型图
底部联系方式 + 社交链接 + 管理后台入口
手机端提示：导航栏在手机上会收起到左上角汉堡菜单（三道横线图标），点击展开导航链接。

3. 查看队员阵容

  1. 点击导航栏「队员阵容」或向下滚动到阵容区域
  2. 队员按位置分组显示：教练、门将、后卫、中场、前锋
  3. 点击卡片正面 → 卡片 3D 翻转，显示背面详细信息（位置、简介、身高体重、擅长位置标签）
  4. 点击「查看详情」 → 打开完整信息弹窗（含 MVP 次数统计）
快速关闭：点击「← 返回」按钮、点击背景遮罩、或按键盘 Esc 键均可关闭详情弹窗。
队长标识：队长（#22 李泽）的卡片边框为金色高亮。

4. 赛程战果与倒计时

倒计时栏

  • 位于赛程区域顶部，显示最近一场即将开赛的比赛
  • 倒计时精确到秒：天、时、分、秒，金色数字实时跳动
  • 比赛进行中时显示绿色「比赛进行中」+ 剩余时间
  • 报名按钮位于倒计时正下方（见第 5 节）

最新战报（左侧）

  • 显示最近 3 场比赛结果
  • 包含：比分、胜负标记（胜/平/负）、进球队员、助攻队员
  • 已投票的比赛右侧显示 ★ MVP 徽章
  • 点击「查看全部战绩」进入完整赛程页面

即将开赛（右侧）

  • 显示未来 3 场比赛
  • 包含：日期、对手、时间、场地、球衣颜色标志

5. 比赛报名 ⭐

这是最常用的功能。每次比赛前需要在网站上报名的，请大家务必关注报名窗口。

报名窗口时间

项目时间
报名开放比赛日前 4 天的晚上 9:00
报名截止比赛日前 2 天的中午 12:00
举例：周日比赛 → 周三晚 9:00 开放报名，周五中午 12:00 截止。
周六比赛 → 周二晚 9:00 开放报名，周四中午 12:00 截止。

报名步骤

1 在首页赛程区域，倒计时栏下方找到报名按钮

2 按钮在不同阶段显示不同内容：

阶段按钮显示可操作
报名未开放距报名 X天XX时XX分XX秒（倒计时）不可点击
报名进行中✋ 立即报名（金色按钮）可点击
报名已截止报名已截止（灰色按钮）不可点击

3 报名开放后，点击金色按钮打开报名面板

4 验证身份：在下拉列表中选择你的名字 → 输入个人 PIN 码 → 点击「验证身份」

5 选择报名类型：

  • 正选（0/13）：前 13 名报名的队员进入正选名单
  • 候补（0/4）：正选满员后自动进入候补队列

6 报名成功后显示「已报名 · 正选」或「已报名 · 候补」状态标签

请假

如果报名后临时有事不能参加：

  1. 在报名面板中点击「我要请假」
  2. 选择请假原因：临时有事 / 身体不适 / 工作加班 / 家庭事务 / 其他（可自由输入）
  3. 确认后自动取消报名，候补队列第一位自动递补为正选
  4. 截止前可以重新报名

注意事项

  • 正选 13 人，候补 4 人，先报先得
  • 截止前 2 小时若报名人数不足 13 人，按钮下方会出现红色提醒
  • 报名记录在截止前可随时修改（取消后重新报名）

6. MVP 投票（金球奖）

每场比赛结束后，正选队员可以为当场表现最好的队友投票。累计积分决定「金球奖」得主。

谁可以投票

  • 只有该场比赛的 正选报名队员 可以投票
  • 每人只能投一次，提交后 不可修改

投票步骤

1 在最新战报中找到对应比赛，点击右侧「MVP 投票」按钮

2 验证身份（选择名字 + 输入 PIN 码）

3 选择排名：

  • 第一名（3 分）
  • 第二名（2 分）
  • 第三名（1 分）

4 同一人不能出现在多个排名中（选择后自动禁用）

5 确认投票内容无误后提交

查看结果

  • 比赛结果右上方显示 「★ MVP: 队员名」
  • 鼠标悬停（手机长按）可查看前三名票数分布
  • 个人累计 MVP 次数会显示在队员详情弹窗中

7. 精彩瞬间与相册

照片轮播

  • 首页「精彩瞬间」区域自动轮播比赛照片
  • 点击左右箭头手动切换
  • 手机端支持左右滑动
  • 鼠标悬停时自动播放暂停

相册浏览

  • 点击导航栏「精彩瞬间」打开相册面板
  • 照片按比赛分组（如「今日说法 vs XX 队」）
  • 「比赛瞬间」相册收录所有比赛照片
  • 点击任意照片全屏放大查看
  • 关闭：点击背景遮罩或按 Esc 键

8. 战术板

首页「战术板」区域展示球队 3-3-1 阵型图，各位置颜色：

  • 橙色 = 门将（1人）
  • 蓝色 = 后卫（3人）
  • 绿色 = 中场（3人）
  • 红色 = 前锋（1人）

此为静态展示，仅供参考。

10. 常见问题

Q: 忘记 PIN 码怎么办？

联系队长重置。PIN 码用于验证身份，确保是本人操作。

Q: 报名后怎么知道是否成功？

报名面板中显示你的报名状态（正选/候补），同时首页报名按钮上的数字会更新。

Q: 手机上看加载很慢？

首次访问可能需要几秒加载图片和数据。之后每次打开会使用本地缓存，基本秒开。建议添加到手机主屏幕像 App 一样使用。

Q: 微信里打开能用吗？

微信内置浏览器可正常使用全部功能，包括报名和投票。

Q: 报名截止后还能取消吗？

截止后系统锁定，无法自助取消。如需取消请在微信群里告诉队长。

Q: 别人能看到我的 PIN 码吗？

PIN 码使用 SHA-256 加密传输和验证，任何人（包括队长）都无法看到你的密码明文。

Q: 发现 bug 或功能建议？

告诉队长或副队长，他们会反馈给开发同学。欢迎任何建议！
