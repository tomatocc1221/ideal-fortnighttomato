# 功能需求清单

## 项目概述

**今日说法足球俱乐部**官方网站，单页滚动式展示型网站，面向球迷和潜在队员。

## 功能区块

### 1. 导航栏
- 固定页面顶部，滚动后背景变半透明毛玻璃
- 队徽 + 球队名称 + 菜单项
- 移动端折叠为汉堡菜单
- 点击菜单项平滑滚动到对应区块

### 2. 英雄区（Hero）
- 全屏展示，队徽居中，球队口号
- 向下滚动箭头动画提示
- 深色背景 + 金色光晕纹理

### 3. 球队介绍
- 球队成立背景、建队理念
- 关键数据统计卡片（4项）
- 简约双栏布局

### 4. 队员阵容
- 队员卡片网格：照片/头像、姓名、号码、位置
- 队长特殊标识（金色边框 + 徽章）
- 卡片悬停动画效果
- **点击卡片进入队员详情页**：全屏覆盖层从右侧滑入，展示大头像、号码、姓名、位置、个人简介、身高/体重/入队年份数据、技术特点标签
- 点击返回按钮或按 ESC 键关闭，支持点击空白区域关闭

### 5. 赛程战果
- 左栏：近期比赛结果（胜/平/负颜色区分）
- 右栏：未来赛程预告
- 数据从 JS 数组动态渲染

### 6. 精彩瞬间
- 照片网格展示
- 点击放大灯箱浏览
- 后续替换真实照片

### 7. 页脚
- 球队基本信息 + 联系方式
- 社交媒体入口
- 版权信息

## 非功能需求

- 响应式设计：适配 375px ~ 1440px+ 宽度
- 加载速度快（无外部依赖）
- 纯静态，可直接双击打开浏览
- 后续通过 GitHub Pages 免费部署上线
