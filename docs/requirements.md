# 功能需求清单

## 项目概述

**今日说法足球俱乐部**官方网站 + 球队管理系统，支持球员报名、管理后台、出勤统计。

## 功能区块

### 1. 导航栏
- 固定页面顶部，滚动后背景变半透明毛玻璃
- 队徽 + 球队名称 + 6个菜单项（含战术板）
- 移动端折叠为汉堡菜单（右侧滑入）
- 点击菜单项平滑滚动到对应区块，自动高亮当前区块

### 2. 英雄区（Hero）
- 全屏展示，队徽居中，球队口号"用脚说话，以球会友"
- Canvas 粒子动画（45个金色光点，脉冲 + 发光）
- 向下滚动箭头动画提示（bounce 2s infinite）
- Visible 检测：页面不可见时暂停动画节省性能

### 3. 球队介绍
- 球队成立背景（2025.03.30 重庆溉澜溪体育公园）、建队理念
- 关键数据统计卡片（4项）：建队年份、城市、主场、建队班底
- 数字滚动计数动画（IntersectionObserver 触发）
- 简约双栏布局

### 4. 队员阵容
- 27名队员卡片网格（照片/首字母头像、姓名、号码、位置）
- 队长特殊标识（金色边框 + CAPTAIN 徽章）
- 按位置分组：教练 → 门将 → 后场 → 中场 → 前场，彩色分组标题
- 位置颜色编码：前锋红、中场绿、后场蓝、门将橙
- 3D 翻转卡片（正面基本信息，背面简介 + 查看详情）
- 卡片悬停动画（上浮 -6px + 阴影）
- **点击卡片进入队员详情页**：全屏覆盖层从右侧滑入，大头像、号码、姓名、位置、年龄、入队年份、技术特点标签
- 头像加载：文件查找 `images/players/{号码}.jpg` → IndexedDB 回退 → 首字母占位
- 点击返回按钮 / ESC 键 / 点击空白区域关闭

### 5. 赛程战果
- 比赛倒计时栏（多场比赛切换、比赛中实时状态）
- 左栏：近期比赛结果（胜金/平灰/负红颜色区分，含进球者/助攻者）
- 默认显示最近 5 场，点击「查看全部战绩」展开完整列表
- 右栏：未来赛程预告（含对手、时间、场地）
- 数据源：API 数据优先 + `data/fixtures.js` 静态数据合并
- 比赛报名按钮（跳转报名面板）

### 6. 战术板
- Canvas 绘制足球场（半场草坪视角，含罚球区、球门等）
- 3 种阵型切换：3-2-2（3后卫） / 2-3-2（2后卫） / 3-3-1（3前锋）
- 球员位置实时渲染（号码 + 姓名），按角色着色
- 响应式 Canvas 缩放

### 7. 精彩瞬间
- 首页轮播（自动播放 + 触摸滑动 + 点状指示器）
- 全屏相册覆盖层（按比赛/活动分组）
- 点击放大灯箱浏览（ESC / 点击背景关闭）
- 无照片时显示占位相机图标

### 8. 比赛报名系统
- 球员身份验证（姓名 + 个人密码 SHA256）
- 三种状态：正选（上限14人）/ 候补（上限4人）/ 请假
- 自动补位：请假时候补自动升为正选
- 重新报名时间戳刷新（排到队尾）
- 费用分摊：场地费 ÷ 正选人数 = 人均费用
- 请假原因预设 + 自定义输入
- 报名窗口：赛前 4 天晚 9 点开放，赛前 8 小时截止

### 9. 管理后台
- 管理员密码登录（基于 token 的 sessionStorage）
- **队员管理**：添加/编辑/删除球员，设置个人密码
- **比赛管理**：创建/编辑/删除比赛，设置场地、球衣、费用、人数上限、报名窗口
- **报名记录**：按比赛查看报名名单，状态调整，删除记录
- **出勤统计**：表格展示总场次、正选、候补、请假、出勤率（颜色分级）
- **赛后录入**：选择已结束比赛，录入比分、进球者、助攻者

### 10. 页脚
- 球队基本信息（常规比赛时间、场地）
- 联系方式（微信、邮箱）
- 社交媒体入口（微信、抖音、微博）
- 版权信息 + 版本号 + 管理后台链接

## 非功能需求

- 响应式设计：适配 375px ~ 1440px+ 宽度
- 性能优化：图片压缩 76%、懒加载、IntersectionObserver、requestAnimationFrame
- 无障碍：语义化 HTML、focus-visible 样式、prefers-reduced-motion 适配
- SEO：Open Graph 元数据、结构化描述
- Safari 兼容：伪元素 aspect-ratio 回退、iOS 安全区适配
- 数据持久化：本地 JSON 文件（开发环境）→ Supabase（生产环境）
- 部署目标：GitHub Pages（前端） + Supabase（后端）
