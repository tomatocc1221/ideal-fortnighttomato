# 设计规范

## 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#0d0d0d` | 页面全局底色 |
| 次背景 | `#111111` | 暗色区块交替 |
| 卡片背景 | `#181818` | 队员卡片、统计卡片 |
| 金色主色 | `#c9a961` | 标题、重点、交互高亮 |
| 正文 | `#e0e0e0` | 主要文字 |
| 次要文字 | `#999999` / `#777777` | 辅助信息 |
| 边框 | `#222222` / `#1a1a1a` | 卡片、分割线 |
| 胜利 | `#c9a961` | 金色背景 |
| 平局 | `#888888` | 灰色背景 |
| 失利 | `#c0392b` | 暗红背景 |

## 字体

```css
font-family: -apple-system, BlinkMacSystemFont,
             "PingFang SC", "Microsoft YaHei",
             "Helvetica Neue", sans-serif;
```

- 中文优先苹方（macOS）/ 微软雅黑（Windows）
- 英文/数字使用系统原生 sans-serif
- 不引入外部 Web 字体（保证加载速度）

## 字号层级

| 层级 | 大小 | 用途 |
|------|------|------|
| H1 大标题 | `clamp(3rem, 8vw, 6rem)` | Hero 区队名 |
| H2 区块标题 | `2rem` | 各 section 标题 |
| H3 小标题 | `1.1rem` / `1rem` | 列表标题 |
| 正文 | `1.05rem` / `0.95rem` | 段落、卡片 |
| 辅助文字 | `0.9rem` / `0.85rem` / `0.8rem` | 标签、日期 |
| 数据大字 | `2.6rem` / `2.2rem` | 统计数字、球衣号码 |

## 间距

| 层级 | 数值 | 用途 |
|------|------|------|
| 区块上下 | `100px`（移动端 `64px`） | section padding |
| 卡片间距 | `28px` / `16px` | grid gap |
| 内容区最大宽 | `1100px` | container max-width |
| 内容区水平内边 | `24px` | container padding |

## 圆角与阴影

- 卡片圆角：`8px` ~ `10px`
- 按钮/标签圆角：`4px` ~ `20px`
- 队徽阴影：`drop-shadow(0 0 24px rgba(201, 169, 97, 0.25))`
- 卡片悬停阴影：`0 12px 40px rgba(0,0,0,0.4)`

## 动效

- 渐现动画：`opacity + translateY`，0.6s ease
- 悬停上浮：`translateY(-6px)`，0.3s~0.35s
- 导航背景：0.3s ease
- Hero 箭头弹跳：2s 无限循环

## 风格关键词

**简约、高级、克制** —— 大量留白、少即是多、金色只做点缀不做铺张。
